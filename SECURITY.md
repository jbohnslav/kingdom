# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in kingdom, please report it responsibly.

**Do not open a public issue.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/jbohnslav/kingdom/security/advisories/new) to submit your report. Include:

- A description of the vulnerability
- Steps to reproduce
- Any relevant logs or screenshots

You should receive an acknowledgment within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Supported Versions

Only the latest release is supported with security updates.
