# kd council critique command

Add command for council members to anonymously evaluate each other's responses (inspired by LLM Council Stage 2). Useful for high-stakes decisions.

## Acceptance Criteria

- kd council critique triggers peer evaluation
- Each member receives all responses anonymized ('Response A', 'Response B', etc.)
- Members identify strengths/weaknesses of each response
- Output displayed in Rich panels
- Critiques saved to <run_id>/critiques/ directory
- Optional --run-id flag to critique a previous run
